# Simple JavaScript Projects

A collection of beginner-friendly JavaScript projects to practice core web development concepts like **DOM manipulation**, **event handling**, and **basic logic**. Each project is organized into its own folder for easy navigation.

## Projects

1. **Calculator**  
   A basic calculator that performs addition, subtraction, multiplication, and division.

2. **Digital Clock**  
   A real-time digital clock displaying hours, minutes, and seconds.

3. **Stopwatch**  
   A simple stopwatch with start, stop, and reset functionalities.

4. **Temperature Converter**  
   Converts temperature between Celsius, Fahrenheit, and Kelvin.

## Features

- Interactive JavaScript functionality
- Simple and easy-to-understand code structure
- Great for beginners to practice and learn JavaScript fundamentals

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/USERNAME/simple-js-projects.git
   ```
2. Navigate to the project folder:
   ```bash
   cd simple-js-projects/Calculator
   ```
3. Open the HTML file in a browser:
   ```text
   open index.html
   ```
4. Explore other projects by opening their respective folders.
