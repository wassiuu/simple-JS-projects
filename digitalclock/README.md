# Digital Clock

A real-time digital clock built using JavaScript that displays hours, minutes, and seconds.

## Features

- Displays current time in HH:MM:SS format in Edmonton,Canada
- Updates every second
- Simple and responsive layout

## How to Run

1. Open the `DigitalClock` folder.
2. Open `index.html` in a web browser.
3. Watch the clock update in real time.
