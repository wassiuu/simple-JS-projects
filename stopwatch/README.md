# Stopwatch

A simple stopwatch built using JavaScript with start, stop, and reset functionality.

## Features

- Start, stop, and reset the timer
- Display in HH:MM:SS format
- Lightweight and interactive

## How to Run

1. Open the `Stopwatch` folder.
2. Open `index.html` in a web browser.
3. Use the buttons to control the stopwatch.

## Notes

- Great for practicing timers and event handling in JavaScript.
