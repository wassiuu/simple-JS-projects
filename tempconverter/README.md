# Temperature Converter

A small JavaScript project that converts temperature between Celsius and Fahrenheit.

## Features

- Convert between Celsius and Fahrenheit
- User-friendly input and display
- Real-time calculation

## How to Run

1. Open the `TemperatureConverter` folder.
2. Open `index.html` in a web browser.
3. Enter a temperature and select the units to convert.

## Notes

- Great practice for using JavaScript functions and handling input/output in the DOM.
